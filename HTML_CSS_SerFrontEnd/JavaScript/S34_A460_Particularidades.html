  1. ha 2 tipos de dados possiveis: primitivos (passados por valor) e objetos(passados pro referência)

    • Number

    • null

  2. variaveis podem armazenar valores de tipos diferentes - fracamente tipada
    • let n = 10
    • n = false
    • n = 'texto'
  3. truthy vs falsy - valores como 0, "", NaN, undefined, null e false
    • são avaliados como false, quando convertidos para Boolean
    • todos os outros valores serão true
  4. Curto-circuito - Operador logicos
    • || - será true se alguma das expressões for verdade
    • && - será true se ambas as expressões forem verdadeiras
    • toString() - pode mudar a base do numero - Nos usamos base padrão 10 ou base decimal
      • Base binária
        • 0 -> 0
        • 1 -> 1
        • 2 -> 10
        • 3 -> 11
        • 4 -> 100
        • 5 -> 101
      • Base hexadecimal
        • 0 -> 0
        • 1 -> 1
        • ...
        • 9 -> 9
        • 10 -> A
        • 11 -> B
        • ...
        • 15 -> F
        • 16 -> 10
        • 17 -> 11
        • ...
        • 26 -> 1A
      • n = 10
      • n.toString(2) = 1010
      • n.toString(16) = A

    • Function declaration
      • function ola1(nome){

      • }
    • function expression
      • const ola2 = function (nome){
      • console.log("Ola mundo, function expression:" + nome)
      • }

    • arrow function
      • const ola3 = (nome) => {
      • console.log("Ola mundo, function expression:" + nome)
      • }
      • ola3 ("Ola mundo, arrow function")
      • arrow function em uma unica linha
      • const soma = (n1,n2) => {
      • return n1 + n2
      • }
      • é o mesmo que:
      • const soma = (n1,n2) => n1 + n2
