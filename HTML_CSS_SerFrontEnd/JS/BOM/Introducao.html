Introdução ao BOM

O BOM (Browser Object Model) é um conjunto de objetos que representam o navegador e permitem a interação com ele. Diferente do DOM, que lida com a estrutura do documento HTML, o BOM lida com aspectos do navegador, como janelas, histórico, localização e outros.

Objetos Principais do BOM

  • window: Representa a janela do navegador. É o objeto global em JavaScript e contém propriedades e métodos para manipular a janela.
  • navigator: Fornece informações sobre o navegador, como nome, versão e plataforma.
  • location: Permite acessar e modificar a URL da página atual.
  • history: Permite navegar pelo histórico de páginas visitadas.
  • screen: Fornece informações sobre a tela do dispositivo, como largura e altura.
